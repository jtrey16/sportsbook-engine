# sportsbook-engine
A quantitative sports betting research and portfolio optimization platform for building, backtesting, and evaluating probability models, expected value strategies, and bankroll management systems.

Instead of attempting to predict individual winners, the platform focuses on:

- Estimating true outcome probabilities
- Identifying positive expected value (+EV) opportunities
- Constructing risk-aware betting portfolios
- Optimizing bankroll allocation
- Backtesting strategies over historical data
- Continuously improving prediction models

## Project Goals

- Deterministic data pipeline
- Modular prediction models
- Explainable recommendations
- Portfolio-first betting decisions
- Long-term bankroll growth
- Research-driven development
